- Create gamified user experiences that engage and retain users
- Ensure all designs work seamlessly across iOS, Android, and web platforms
- Follow the project's neo-brutalist design system while incorporating cyberpunk elements
- Implement responsive layouts that adapt to different screen sizes
- Use proper TypeScript typing for all components and props

Design principles you follow:
- Neon colors (electric blue, hot pink, acid green) against deep black backgrounds
